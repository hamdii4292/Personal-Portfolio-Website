# Personal Modern Responsive Portfolio Website

Welcome to the repository of my personal portfolio website. This is a clean, modern, and high-performance single-page portfolio designed to showcase my skills, services, and full-stack projects. It is built entirely from scratch using semantic HTML5, modern CSS layouts (Grid and Flexbox).
## 🚀 Live Demo
Check out the live deployment here: https://hammad-rasheed.vercel.app/
## ✨ Key Features
* Sticky Navigation Bar: Fluid header layout that stays visible while scrolling, transforming into an animated hamburger menu on mobile screens.
* Dynamic Hero Section: Eye-catching design utilizing custom CSS background gradients, interactive social media rings, and a neon-glow circular profile frame.
* Modular Layouts: Clean separation of sections ("About Me", "My Services", "Contact") leveraging Flexbox for alignment and CSS Grid for responsive cards.
* Scroll-Triggered Animations: High-performance fade-and-rise entrance effects built using the modern native JavaScript Intersection Observer API (no bulky external animation frameworks required).
* Fully Mobile-Responsive: Built from a mobile-first mindset with tailored media queries ensuring optimal presentation on smartphones, tablets, and large computer monitors.
* Functional Actions: Directly supports resume downloads and integrated social connectivity (LinkedIn, GitHub, Instagram) opening natively in separate browser tabs.

## 🛠️ Technologies Used
* HTML5 – Structured semantic web layout.
* CSS3 – Custom variables (:root), Flexbox, CSS Grid, media queries, linear gradients, and keyframe transitions.
* Font Awesome – Vector iconography for services and social networks.
* Google Fonts – Premium topography pairing using the 'Poppins' font family.
